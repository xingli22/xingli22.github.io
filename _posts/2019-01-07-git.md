## 设置git

>$ git config --global user.name "xxx"

>$ git config --global user.email "xxx@outlook.com"


查看git配置：

'''
$git config --lis
'''

## 生成SSH密钥
 
查看是否已经有了ssh密钥

>cd ~/.ssh
 
生成密钥

>$ ssh-keygen -t rsa -C "xxx@outlook.com"
